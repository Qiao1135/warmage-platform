# 暖圈 - 老年人社交平台规范文档

## 1. Concept & Vision

暖圈是一款专为银发族设计的社交与陪伴平台，以"温暖相伴、暖心互动"为核心理念。产品针对老年人视力下降、听力减退、触屏操作困难等生理特点，采用大字体、大按钮、高对比度的极简设计，让科技变得温暖易用。通过语音发帖、AI陪聊、志愿者关怀等功能，让老年人在数字时代也能轻松社交、享受陪伴。

## 2. Design Language

### 2.1 Aesthetic Direction
温暖治愈系设计，以柔和的暖色调为主，营造温馨、亲切、安全的使用氛围。视觉语言借鉴传统中国风的温馨感与现代扁平设计的简洁易用。

### 2.2 Color Palette
```
Primary:     #FF6B6B (暖心红 - 代表热情与关怀)
Secondary:   #4ECDC4 (清新青 - 平衡与活力)
Accent:      #FFE66D (阳光黄 - 快乐与温暖)
Background:  #FFF9F0 (奶白色 - 温馨底色)
Surface:     #FFFFFF (纯白 - 卡片与内容区)
Text-Primary:#333333 (深灰 - 主要文字)
Text-Secondary:#666666 (中灰 - 次要文字)
Success:     #2ECC71 (满意绿)
Warning:     #F39C12 (提醒橙)
Error:       #E74C3C (警示红)
```

### 2.3 Typography
- **主标题**: 思源黑体 / Noto Sans SC, 24px-32px, Bold
- **正文大字**: 思源黑体 / Noto Sans SC, 18px-20px, Regular
- **按钮文字**: 思源黑体 / Noto Sans SC, 18px, Medium
- **辅助说明**: 思源黑体 / Noto Sans SC, 14px-16px, Regular
- **识别结果**: 思源黑体 / Noto Sans SC, 26px, Regular (语音转文字展示)

### 2.4 Spatial System
- 基础间距单位: 8px
- 卡片内边距: 16px-24px
- 卡片间距: 12px
- 按钮高度: 最小56px（大触摸目标）
- 图标尺寸: 最小48x48px
- 安全点击区域: 44x44px

### 2.5 Motion Philosophy
- **反馈动画**: 按钮点击缩放0.95，轻微弹性
- **状态转换**: 300ms ease-out 平滑过渡
- **录音波形**: 实时音量映射，60fps流畅动画
- **点赞特效**: 心形放大+粒子散开，持续600ms
- **页面切换**: 左右滑动，400ms ease-in-out

### 2.6 Visual Assets
- **图标库**: 使用圆润、友好风格的图标，便于识别
- **插画风格**: 温暖的场景插画，如家庭、自然、社交
- **头像**: 默认使用温暖的渐变头像，可自定义
- **装饰元素**: 圆润的形状，柔和的阴影

## 3. Layout & Structure

### 3.1 整体架构
采用底部Tab导航，适合老年人单手操作：
```
┌─────────────────────────┐
│      Header (标题)       │
├─────────────────────────┤
│                         │
│      Content Area       │
│      (可滚动内容)         │
│                         │
├─────────────────────────┤
│   🏠  📝  💬  👤         │
│   首页  发帖  消息  我的    │
└─────────────────────────┘
```

### 3.2 页面结构

**首页 (暖圈)**
- 顶部: 问候语 + 日期 + 天气
- 话题标签横向滚动
- 帖子流瀑布式布局

**发帖页 (暖圈)**
- 录音按钮居中（大号绿色）
- 音量波形可视化区
- 识别结果展示区
- 图片选择/预览区

**暖伴页**
- AI陪聊入口
- 志愿者预约入口
- 子女周报入口
- 每日推送设置

**我的页面**
- 头像+昵称
- VIP状态
- 功能入口列表

## 4. Features & Interactions

### 4.1 暖圈模块

#### 4.1.1 语音发帖
**核心流程:**
1. 用户点击/长按绿色麦克风按钮
2. 屏幕显示动态音量波形图
3. 松手后自动转文字
4. 展示识别结果（大号26px字体）
5. 用户可重新录制或添加图片
6. 点击发布提交

**交互细节:**
- 按下: 按钮变色 + 轻微震动 + 波形开始
- 录音中: 波形实时响应音量，最长60秒
- 松手: 震动反馈 + 转圈加载 + 显示结果
- 重新说: 按钮变橙色，点击清空重录
- 添加图片: 最多9张，九宫格排列
- 发布: 按钮渐变背景，发布成功弹窗

#### 4.1.2 话题广场
**话题标签:**
- 广场舞、养花、书法、下棋、养生、旅游、美食、摄影
- 横向滚动，当前选中高亮

**帖子列表:**
- 按时间倒序
- 卡片式布局
- 显示: 头像、昵称、时间、内容、图片、点赞数、评论数
- 点击进入详情页

**关注话题:**
- 点击"关注"按钮
- 按钮变为"已关注"
- 新帖子优先推送

#### 4.1.3 评论与互动
**一键暖心评论 (20条):**
```
😊 写得真好    🤩 太厉害了    ❤️ 我也喜欢这个
😄 看着就开心  👏 您真棒      🙏 谢谢分享
📚 学到了      🌸 太美了      👍 给您点赞
💖 心情都变好了  🌟 真棒      😊 支持
🎉 太好了      💪 加油        ☀️ 阳光
🌈 真美好      🎶 美好的一天  🍀 祝福
🌺 好看        💐 送给你
```

**交互:**
- 点击短语直接发送
- 大号按钮，便于点击
- 同时保留手写/语音输入入口

**点赞:**
- 双击帖子任意区域触发
- 心形变红 + 放大动画
- 播放提示音
- 语音播报"收到一个赞"

### 4.2 暖伴模块

#### 4.2.1 AI语音陪聊
**界面:**
- 对话气泡区域
- 底部大号麦克风按钮
- 当前对话状态指示

**对话流程:**
1. 按住麦克风说话
2. 松手发送语音
3. AI处理并生成回复
4. 语音合成播放回复
5. 自动生成对话气泡

**话题预设:**
- 往事回忆、家庭琐事
- 健康咨询、兴趣爱好
- 今日天气、日常问候

**限制与提醒:**
- 免费用户: 每日5分钟
- VIP用户: 不限时长
- 连续30分钟: 弹休息提醒

#### 4.2.2 志愿者电话陪聊
**预约极简设计:**
- 三个大按钮: "今天" "明天" "后天"
- 点击后自动匹配
- 预约成功语音提示

**预约成功反馈:**
- 语音: "已帮您约好了明天下午3点的电话"
- 显示预约信息卡片

**服务评价:**
- 三个大按钮: "满意" "一般" "不满意"
- 通话结束后弹出

**限制:**
- 每周1次免费
- 超次: 每次5元
- 支持子女代付

#### 4.2.3 子女周报
**免费版:**
- 活跃天数
- 发帖数量

**VIP版:**
- 高频词云
- 情绪趋势图
- 异常提醒
- AI个性化建议

#### 4.2.4 每日暖心语音推送
**推送时间:** 每天早上8点

**内容个性化:**
- 天气信息
- 节假日问候
- 兴趣偏好

**VIP声音定制:**
- 慈祥奶奶声
- 活泼孙子声
- 知性老师声

### 4.3 支付与订阅系统

#### 4.3.1 一键代付
**流程:**
1. 老人点击开通VIP
2. 选择"让子女帮我付"
3. 生成包含信息的二维码
4. 截图发送给子女
5. 子女扫码支付
6. 自动开通VIP

#### 4.3.2 连续订阅提醒
- 开通时确认续费提醒
- 到期前3天提醒
- 到期前1天再次提醒
- 每3个月满意度调查

## 5. Component Inventory

### 5.1 底部导航栏 (BottomNav)
- 4个Tab: 首页、发帖、消息、我的
- 当前选中高亮(Primary色)
- 图标尺寸: 48x48px
- 底部安全区域适配

### 5.2 语音录音按钮 (RecordButton)
- 默认: 绿色圆形，56px高度
- 按下: 颜色加深，缩小至0.95
- 录音中: 动态脉冲效果
- 禁用: 灰色，opacity: 0.5

### 5.3 音量波形 (VolumeWave)
- 32根波形柱
- 高度范围: 4px - 100px
- 颜色渐变: 绿色 -> 黄色 -> 红色
- 60fps动画更新

### 5.4 暖心评论按钮 (WarmCommentBtn)
- 圆角按钮，padding: 12px 20px
- 背景: #FFF0F0
- 边框: 1px solid #FFBDBD
- 点击: scale(0.95) + 震动

### 5.5 帖子卡片 (PostCard)
- 白色背景，圆角16px
- 阴影: 0 2px 8px rgba(0,0,0,0.08)
- 图片九宫格: 3列，间距4px

### 5.6 话题标签 (TopicTag)
- 胶囊形状，padding: 8px 16px
- 未选中: 浅灰背景
- 选中: Primary色背景，白色文字

### 5.7 AI对话气泡 (ChatBubble)
- 用户: 右侧，浅绿背景
- AI: 左侧，白色背景
- 最大宽度: 75%
- 圆角: 16px

### 5.8 预约日期按钮 (DateButton)
- 大尺寸: 高度72px
- 日期文字: 24px Bold
- 选中: Primary渐变背景

### 5.9 VIP状态卡片 (VIPCard)
- 渐变背景: #FF6B6B -> #FF8E8E
- 金色VIP徽章
- 到期日期显示

### 5.10 二维码卡片 (QRCodeCard)
- 白色卡片，圆角24px
- 包含: 昵称、价格、有效期
- 二维码居中显示

## 6. Technical Approach
